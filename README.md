## Usage

All components use **Tailwind CSS** — copy any component from the [live demo](https://ayushkumar372.github.io/uidrop-components) directly into your project.

```bash
# Install Tailwind CSS (if not already)
npm install -D tailwindcss
npx tailwindcss init
```
